# 常见问题解答 (FAQ)

以下是使用 PoSDK 过程中常见的问题及解答。如果您的问题未在此列出，请查阅详细文档或通过 GitHub 提交 issue。

## 安装相关问题

### Q: 如何解决依赖项安装错误？

A: 大多数依赖项安装错误可以通过以下步骤解决：
- 确保您的系统满足最低版本要求
- 使用包管理器（如apt、brew）安装依赖
- 对于特定版本需求，考虑从源代码编译

### Q: 在macOS/Windows上能否使用PoSDK？

A: 当前版本主要支持Linux环境，我们正在努力提供对macOS和Windows的完整支持。请关注项目更新。

## 插件开发问题

### Q: 如何创建自定义的数据插件？

A: 请参考"基础开发"章节中的"数据插件"部分，其中提供了详细的开发步骤和示例代码。

### Q: 为什么我的插件无法被系统识别？

A: 常见原因包括：
- 插件注册宏未正确使用
- GetType()返回值与注册的类型字符串不一致
- 插件库未被正确加载到系统中
