# po_core 构建环境与版本信息

本文档说明 PoCore核心库的自动工作流编译环境配置和依赖版本要求。

## 概述

po_core 是 PoSDK 的核心库，通过自动工作流在不同平台上构建和发布。本文档列出了：

- po_core 支持的构建平台
- 构建工具版本要求
- 依赖库版本信息

## 构建平台支持

po_core 目前支持以下平台的自动构建：

| 平台       | 架构   | 版本                       | 状态       |
| ---------- | ------ | -------------------------- | ---------- |
| **Ubuntu** | x86_64 | 24.04.3 LTS (Noble Numbat) | ✅ 完全支持 |
| **Ubuntu** | arm64  | 24.04.3 LTS (Noble Numbat) | ✅ 完全支持 |
| **macOS**  | arm64  | 15.3+                      | ✅ 完全支持 |

## 构建工具版本要求

### 编译器

| 平台         | 编译器    | 最低版本 | 推荐版本 | 说明                        |
| ------------ | --------- | -------- | -------- | --------------------------- |
| Ubuntu 24.04 | GCC       | 11.0     | 13.3.0   | 支持 C++17                  |
| macOS        | GCC/Clang | 13.0     | 16.0.0   | Apple Clang 或 Homebrew GCC |

### CMake

| 平台     | 最低版本 | 推荐版本 | 说明                              |
| -------- | -------- | -------- | --------------------------------- |
| 所有平台 | 3.28     | 3.28+    | 某些依赖库（如 GLOMAP）需要 3.28+ |

## 依赖库版本信息

### 关键依赖库

| 依赖库           | Ubuntu 24.04                     | macOS                            |
| ---------------- | -------------------------------- | -------------------------------- |
| **Suite-Sparse** | system_installed                 | 7.11.0 (Homebrew)                |
| **METIS**        | system_installed                 | 5.1.0 (Homebrew)                 |
| **OpenSSL**      | 3.0.13 (静态链接)                | 3.6.0 (静态链接)                 |
| **Ceres Solver** | 2.2.0-local (静态链接，miniglog) | 2.2.0-local (静态链接，miniglog) |

### 其他依赖库

| 依赖库     | Ubuntu 24.04         | macOS           | 说明                                   |
| ---------- | -------------------- | --------------- | -------------------------------------- |
| **CURL**   | 8.5.0                | 8.7.1           | 动态链接                               |
| **OpenMP** | libgomp (gcc-13.3.0) | libomp (21.1.3) | 动态链接                               |
| **OpenCV** | 4.x-local            | 4.x-local       | 本地构建，路径：`opencv/install_local` |
